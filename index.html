Projects

MovieClique

"MovieClique" is an innovative and user-friendly movie application that leverages the power of the TMDB API to provide an extensive collection of movies. The application allows users to explore and discover movies based on specific properties, manage favorites, search for movies, and sort them by rating.

project's features:

I. TMDB API Integration: MovieClique seamlessly integrates with the TMDB (The Movie Database) API, ensuring access to a vast library of movies. The API connection allows the application to fetch up-to-date movie information, including titles, genres, release dates, ratings, and more

II. Movie Search: The application offers a powerful search functionality that allows users to find movies quickly. By entering keywords, movie titles, or actors' names, users can instantly retrieve relevant movie suggestions, facilitating a smooth browsing experience.

III. Sorting by Rating: To assist users in finding highly-rated movies, MovieClique provides the option to sort the movie collection based on ratings. Users can sort movies in ascending or descending order, allowing them to explore the most popular or critically acclaimed titles effortlessly.

IV. Favorite List: MovieClique enables users to create and manage a personalized list of favorite movies. With just a single click, users can add or remove movies from their favorites list, making it convenient to keep track of films they love..
